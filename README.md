A powerful and flexible tool for automated generation of various answer sheets, focusing on ease of use for educational and personal needs.

## ‚ú® About the Project

The `answer-sheet` project is designed to simplify the process of creating necessary answer sheets for exams, exercises, and personal requirements. This system allows users to quickly and easily generate their desired answer sheets in three main models through a web-based user interface.

## üöÄ Key Features

The project is capable of generating three different types of answer sheets:

1.  **Standard Answer Sheet:** Common and general-purpose answer sheets.
2.  **Lined Answer Sheet:** Answer sheets featuring guidance lines for descriptive responses.
3.  **Innovative/Custom Answer Sheet:** Special and proprietary designs for specific needs.

## üõ†Ô∏è Setup and Running

Running this project does not require the installation of external dependencies; it is executable with only one prerequisite installed.

### 1. Prerequisites

* **Python:** Must be installed on your system.

### 2. How to Run

Follow these steps to set up and run the `answer-sheet` project:

#### A) Activate the Project Environment

In your terminal, first execute the activation script for the project environment:

```bash
./answer/scripts/activate
````

#### B) Run the Server

After activating the environment, start the development server using the Django command:

```bash
python manage.py runserver
```

### 3\. Access

Once the server runs successfully, the project will typically be accessible in your browser at the following address:

`http://127.0.0.1:8000/`
